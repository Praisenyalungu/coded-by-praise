Lesson 4: JavaScript Basics

JavaScript adds interactivity and logic to your web pages. You can use it to handle user input, manipulate HTML, and create dynamic content.

Key Concepts

  • Variables & Data Types: Already learned, now used in JS.
  • Functions: Reusable blocks of code.
  • Events: Respond to user actions like clicks or typing.
  • DOM Manipulation: Change HTML content dynamically with JS.

Example Code

// Function to greet user
function greet() {
  const name = prompt("Enter your name:");
  alert("Hello, " + name + "!");
}
