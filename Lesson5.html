Lesson 5: Mini Project

Now it’s time to apply everything you’ve learned. In this mini project, you will create a simple interactive webpage using HTML, CSS, and JavaScript.

Project Steps

  1. Create an HTML page with a heading and a paragraph.
  2. Use CSS to style your page (colors, fonts, spacing).
  3. Add a button that triggers a JavaScript alert saying “Well done!” when clicked.
