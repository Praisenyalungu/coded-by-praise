Lesson 2: HTML Basics

HTML (HyperText Markup Language) is the foundation of the web. It structures your content so browsers know what to display.

Key Concepts

  • Elements: The building blocks of HTML, like <p>, <h1>, <div>.
  • Tags: Wrap content to define its purpose or style.
  • Attributes: Provide extra information, e.g., href in links, src in images.
  • Structure: Basic HTML page includes <!DOCTYPE html>, <html>, <head>, and <body>.

Example Code

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>
