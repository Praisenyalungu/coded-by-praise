Lesson 3: CSS Basics

CSS (Cascading Style Sheets) is used to style and visually enhance your HTML content. It controls colors, layouts, fonts, spacing, and more.

Key Concepts

  • Selectors: Target HTML elements (e.g., p, #id, .class).
  • Properties: Define what to change (e.g., color, font-size, background-color).
  • Values: Set the appearance (e.g., red, 16px).
  • Inline vs Internal vs External CSS: Ways to apply CSS to HTML.

Example Code

<!DOCTYPE html>
<html>
  <head>
    <style>
      p {
        color: blue;
        font-size: 18px;
      }
      .highlight {
        background-color: yellow;
      }
    </style>
  </head>
  <body>
    <p>This is a styled paragraph.</p>
    <p class="highlight">This paragraph is highlighted!</p>
  </body>
</html>
