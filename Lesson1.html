Lesson 1: Variables & Data Types

Welcome to your first coding lesson! In this lesson, you'll learn what variables are, the different data types in JavaScript, and how to use them.

Key Concepts

  • Variables: Containers for storing data values.
  • Data Types: Examples include strings, numbers, booleans, and arrays.
  • Declaration: Using let, const, and var to create variables.

Example Code

// Declaring variables
let name = "Praise";
const age = 23;
var isLearning = true;

console.log(name, age, isLearning);
